# 數位銀行服務說明

## 網路銀行

### 功能一覽

| 功能類別 | 可使用功能 |
|---------|-----------|
| 帳戶管理 | 餘額查詢、交易明細、存摺電子化 |
| 信用卡 | 帳單查詢、繳費、分期申請、額度查詢 |
| 轉帳匯款 | 本行轉帳、跨行轉帳、約定帳號設定 |
| 存款 | 定期存款申請/解約、利率查詢 |
| 貸款 | 還款查詢、提前還款申請 |
| 設定 | 密碼修改、交易限額設定、通知設定 |

### 使用前準備

1. 持身分證至任一分行申請「網路銀行使用權限」
2. 設定網路銀行帳號與密碼
3. 可選擇性申請「數位憑證」以解鎖進階功能（如：提高轉帳額度）

### 登入方式

- 帳號密碼登入
- 晶片金融卡 + 讀卡機登入（最高安全等級）
- 手機 OTP 驗證登入

---

## 行動 APP（XX Bank APP）

### 下載方式
- iOS：App Store 搜尋「XX Bank」
- Android：Google Play 搜尋「XX Bank」
- 目前版本：v5.2（建議保持最新版本）

### 行動 APP 獨有功能
- **掃碼付款**：QR Code 轉帳 / 繳費
- **生物辨識登入**：Face ID / 指紋辨識
- **推播通知**：消費即時通知、帳單提醒
- **附近分行 / ATM 查詢**：GPS 定位最近據點
- **電子對帳單**：無紙化帳單，可匯出 PDF

### APP 快速轉帳
- 不需登入，可設定常用帳號快速轉帳
- 每日轉帳上限：NT$5 萬（低於網路銀行，以保障安全）

## 忘記密碼處理

### 網路銀行密碼重設

**方式一：線上重設（需有手機驗證）**
1. 網路銀行登入頁 → 點「忘記密碼」
2. 輸入帳號 + 身分證字號
3. 收取簡訊 OTP 驗證碼（發至綁定手機）
4. 設定新密碼

**方式二：臨櫃辦理**
- 持身分證至任一分行
- 當場重設密碼，即時生效

### 提款卡密碼重設
- 只能臨櫃辦理，無法線上重設
- 持身分證至任一分行，約 10 分鐘完成

## 帳戶安全設定

### 建議開啟的安全功能
- **消費即時通知**：每筆消費立即 APP 推播（免費）
- **登入異地通知**：非常用裝置登入時發送警示
- **交易確認機制**：大額轉帳需二次驗證（OTP）

### 可疑交易處理
如發現未授權交易，請立即：
1. 撥打 24 小時客服 0800-XXX-XXX 暫停帳戶
2. 或透過 APP → 安全中心 → 緊急凍結帳戶
3. 至警察局報案，取得報案證明
4. 攜帶報案證明至分行辦理正式掛失與申訴

---

## 電子帳單

- **申請方式**：網路銀行 / APP → 帳戶設定 → 電子帳單
- 申請後停止寄送紙本帳單（下期起生效）
- 帳單發送至登記電子信箱，同時可於 APP 查閱
- 可隨時改回紙本帳單

## 常見問題

**Q：APP 登入一直顯示「裝置未認證」怎麼辦？**
A：請至網路銀行 → 安全設定 → 裝置管理，移除舊裝置後重新在新手機完成認證，或聯絡客服協助。

**Q：收不到 OTP 簡訊怎麼辦？**
A：請確認手機號碼是否為銀行登記號碼；若更換手機號碼，需臨櫃更新。也可嘗試切換 Wi-Fi / 行動數據後重新發送。

**Q：網路銀行轉帳被鎖定了怎麼辦？**
A：多次輸入錯誤交易密碼會暫時鎖定轉帳功能，請撥打客服 0800-XXX-XXX 解除，或次日 00:00 自動重置（視鎖定類型而定）。
